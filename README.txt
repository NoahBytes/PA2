HOW TO RUN THE SIMULATION:

METHOD 1 -- Using the script
If you want to run all 21 arrival rates seamlessly, compile the runsimulation.py file.
NOTE: Must be in the same directory as PA2.py

INSTRUCTIONS TO EXECUTE:
python runsimulations.py


METHOD 2 -- Command line compilation
To enter manual commands, you can compile the PA2.py file. Fill in the arrival times
and service times as you wish.

INSTRUCTIONS TO EXECUTE:
python PA2.py <integer arrival time> <float service time>
